			<h1 id="firstHeading" class="firstHeading" lang="en">Biological system</h1>
									<div id="bodyContent" class="mw-body-content">
									<div id="siteSub">From Wikipedia, the free encyclopedia</div>
								<div id="contentSub"></div>
												<div id="jump-to-nav" class="mw-jump">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
				</div>
				<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><p>A <b>biological system</b> is a complex <a href="/wiki/Biological_network" title="Biological network">network</a> of biologically relevant <a href="/wiki/Entities" class="mw-redirect" title="Entities">entities</a>. As biological organization spans several scales, examples of biological systems are <a href="/wiki/Populations" class="mw-redirect" title="Populations">populations</a> of <a href="/wiki/Organisms" class="mw-redirect" title="Organisms">organisms</a>, or on the <a href="/wiki/Organ_(anatomy)" title="Organ (anatomy)">organ</a>- and <a href="/wiki/Tissue_(biology)" title="Tissue (biology)">tissue</a> scale in <a href="/wiki/Mammal" title="Mammal">mammals</a> and other animals, the <a href="/wiki/Circulatory_system" title="Circulatory system">circulatory system</a>, the <a href="/wiki/Respiratory_system" title="Respiratory system">respiratory system</a>, the <a href="/wiki/Nervous_system" title="Nervous system">nervous system</a>, etc.</p>
<p>On the <a href="/wiki/Micrometre" title="Micrometre">micro</a> to the <a href="/wiki/Nanometre" title="Nanometre">nano</a>scopic scale, examples of biological systems are <a href="/wiki/Cells" class="mw-redirect" title="Cells">cells</a>, <a href="/wiki/Organelles" class="mw-redirect" title="Organelles">organelles</a>, macromolecular complexes and regulatory pathways.</p>
<p>A biological system is not to be confused with a <a href="/wiki/Living_systems" title="Living systems">living system</a>, which is commonly referred to as <a href="/wiki/Life" title="Life">life</a>. For further information see e.g. <a href="/wiki/Definition_of_life" class="mw-redirect" title="Definition of life">definition of life</a> or <a href="/wiki/Synthetic_biology" title="Synthetic biology">synthetic biology</a>.</p>
<h2><span class="mw-headline" id="Organ_and_tissue_systems">Organ and tissue systems</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Biological_system&amp;action=edit&amp;section=1" title="Edit section: Organ and tissue systems">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:302px;"><a href="/wiki/File:TE-Nervous_system_diagram.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b2/TE-Nervous_system_diagram.svg/300px-TE-Nervous_system_diagram.svg.png" width="300" height="352" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b2/TE-Nervous_system_diagram.svg/449px-TE-Nervous_system_diagram.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b2/TE-Nervous_system_diagram.svg/599px-TE-Nervous_system_diagram.svg.png 2x" data-file-width="766" data-file-height="900" /></a>
<div class="thumbcaption">
<div class="magnify"><a href="/wiki/File:TE-Nervous_system_diagram.svg" class="internal" title="Enlarge"></a></div>
An example of a system: The <a href="/wiki/Brain" title="Brain">brain</a>, the <a href="/wiki/Cerebellum" title="Cerebellum">cerebellum</a>, the <a href="/wiki/Spinal_cord" title="Spinal cord">spinal cord</a>, and the <a href="/wiki/Nerve" title="Nerve">nerves</a> are the four basic components of the <a href="/wiki/Nervous_system" title="Nervous system">nervous system</a>.</div>
</div>
</div>
<p>These specific systems are widely studied in <a href="/wiki/Human_anatomy" class="mw-redirect" title="Human anatomy">Human anatomy</a>. <i>"Human" systems</i> are also present in many other animals.</p>
<ul>
<li><a href="/wiki/Circulatory_system" title="Circulatory system">Circulatory system</a>: pumping and channeling <a href="/wiki/Blood" title="Blood">blood</a> to and from the body and <a href="/wiki/Lungs" class="mw-redirect" title="Lungs">lungs</a> with <a href="/wiki/Heart" title="Heart">heart</a>, blood and <a href="/wiki/Blood_vessel" title="Blood vessel">blood vessels</a>.</li>
<li><a href="/wiki/Integumentary_system" title="Integumentary system">Integumentary system</a>: <a href="/wiki/Skin" title="Skin">skin</a>, <a href="/wiki/Hair" title="Hair">hair</a>, <a href="/wiki/Fat" title="Fat">fat</a>, and <a href="/wiki/Nail_(anatomy)" title="Nail (anatomy)">nails</a>.</li>
<li><a href="/wiki/Skeletal_system" class="mw-redirect" title="Skeletal system">Skeletal system</a>: structural support and protection with <a href="/wiki/Bone" title="Bone">bones</a>, <a href="/wiki/Cartilage" title="Cartilage">cartilage</a>, <a href="/wiki/Ligament" title="Ligament">ligaments</a> and <a href="/wiki/Tendon" title="Tendon">tendons</a>.</li>
<li><a href="/wiki/Reproductive_system" title="Reproductive system">Reproductive system</a>: the <a href="/wiki/Sex_organs" class="mw-redirect" title="Sex organs">sex organs</a>, such as <a href="/wiki/Ovary" title="Ovary">ovaries</a>, <a href="/wiki/Fallopian_tube" title="Fallopian tube">fallopian tubes</a>, <a href="/wiki/Uterus" title="Uterus">uterus</a>, <a href="/wiki/Vagina" title="Vagina">vagina</a>, <a href="/wiki/Mammary_gland" title="Mammary gland">mammary glands</a>, <a href="/wiki/Testes" class="mw-redirect" title="Testes">testes</a>, <a href="/wiki/Vas_deferens" title="Vas deferens">vas deferens</a>, <a href="/wiki/Seminal_vesicle" title="Seminal vesicle">seminal vesicles</a> and <a href="/wiki/Prostate" title="Prostate">prostate</a></li>
<li><a href="/wiki/Digestive_system" class="mw-redirect" title="Digestive system">Digestive system</a>: <a href="/wiki/Digestion" title="Digestion">digestion</a> and processing food with <a href="/wiki/Salivary_gland" title="Salivary gland">salivary glands</a>, <a href="/wiki/Oesophagus" class="mw-redirect" title="Oesophagus">oesophagus</a>, <a href="/wiki/Stomach" title="Stomach">stomach</a>, <a href="/wiki/Liver" title="Liver">liver</a>, <a href="/wiki/Gallbladder" title="Gallbladder">gallbladder</a>, <a href="/wiki/Pancreas" title="Pancreas">pancreas</a>, <a href="/wiki/Intestine" class="mw-redirect" title="Intestine">intestines</a>, <a href="/wiki/Rectum" title="Rectum">rectum</a> and <a href="/wiki/Anus" title="Anus">anus</a>.</li>
<li><a href="/wiki/Urinary_system" title="Urinary system">Urinary system</a>: <a href="/wiki/Kidney" title="Kidney">kidneys</a>, <a href="/wiki/Ureter" title="Ureter">ureters</a>, <a href="/wiki/Urinary_bladder" title="Urinary bladder">bladder</a> and <a href="/wiki/Urethra" title="Urethra">urethra</a> involved in fluid balance, <a href="/wiki/Electrolyte" title="Electrolyte">electrolyte</a> balance and excretion of <a href="/wiki/Urine" title="Urine">urine</a>.</li>
<li><a href="/wiki/Respiratory_system" title="Respiratory system">Respiratory system</a>: the organs used for <a href="/wiki/Breathing" title="Breathing">breathing</a>, the <a href="/wiki/Human_pharynx" class="mw-redirect" title="Human pharynx">pharynx</a>, <a href="/wiki/Larynx" title="Larynx">larynx</a>, <a href="/wiki/Bronchi" class="mw-redirect" title="Bronchi">bronchi</a>, <a href="/wiki/Lung" title="Lung">lungs</a> and <a href="/wiki/Thoracic_diaphragm" title="Thoracic diaphragm">diaphragm</a>.</li>
<li><a href="/wiki/Endocrine_system" title="Endocrine system">Endocrine system</a>: communication within the body using <a href="/wiki/Hormone" title="Hormone">hormones</a> made by <a href="/wiki/Endocrine_gland" title="Endocrine gland">endocrine glands</a> such as the <a href="/wiki/Hypothalamus" title="Hypothalamus">hypothalamus</a>, <a href="/wiki/Pituitary" class="mw-redirect" title="Pituitary">pituitary</a> gland, <a href="/wiki/Pineal_body" class="mw-redirect" title="Pineal body">pineal body</a> or pineal gland, <a href="/wiki/Thyroid" title="Thyroid">thyroid</a>, <a href="/wiki/Parathyroid" class="mw-redirect" title="Parathyroid">parathyroid</a> and <a href="/wiki/Adrenal" class="mw-redirect" title="Adrenal">adrenals</a>, i.e., adrenal glands.</li>
<li><a href="/wiki/Immune_system" title="Immune system">Immune system</a>: protects the organism from foreign bodies
<ul>
<li><a href="/wiki/Lymphatic_system" title="Lymphatic system">Lymphatic system</a>: structures involved in the transfer of <a href="/wiki/Lymph" title="Lymph">lymph</a> between <a href="/wiki/Tissue_(biology)" title="Tissue (biology)">tissues</a> and the <a href="/wiki/Blood_stream" class="mw-redirect" title="Blood stream">blood stream</a>; includes the lymph and the <a href="/wiki/Lymph_node" title="Lymph node">nodes</a> and <a href="/wiki/Lymph_vessel" class="mw-redirect" title="Lymph vessel">vessels</a>. The lymphatic system includes functions including immune responses and development of antibodies.</li>
</ul>
</li>
<li><a href="/wiki/Muscular_system" title="Muscular system">Muscular system</a>: allows for manipulation of the environment, provides locomotion, maintains posture, and produces heat. Includes <a href="/wiki/Skeletal_muscles" class="mw-redirect" title="Skeletal muscles">skeletal muscles</a>, <a href="/wiki/Smooth_muscles" class="mw-redirect" title="Smooth muscles">smooth muscles</a> and <a href="/wiki/Cardiac_muscle" title="Cardiac muscle">cardiac muscle</a>.</li>
<li><a href="/wiki/Nervous_system" title="Nervous system">Nervous system</a>: collecting, transferring and processing information with <a href="/wiki/Brain" title="Brain">brain</a>, <a href="/wiki/Spinal_cord" title="Spinal cord">spinal cord</a> and <a href="/wiki/Peripheral_nervous_system" title="Peripheral nervous system">peripheral nervous system</a>.</li>
</ul>
